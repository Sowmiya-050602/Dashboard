# Car Sales Dashboard 

## 📊 Overview

This project is an interactive **Car Sales Dashboard** built using Microsoft Power BI to analyze and visualize sales performance across different dimensions such as time, region, vehicle type, and dealer performance.


##  Key Features

* YTD Sales Analysis – Track total sales, average price, and cars sold
* Monthly Trends – Monitor sales fluctuations over time
* Regional Insights – Visual map showing sales distribution by dealer region
* Body Style Breakdown – Compare performance across SUV, Sedan, Hatchback, etc.
* Color Analysis – Understand customer preferences by car color
* Company-wise Performance – Analyze sales by different car manufacturers


## Dashboard Insights

* Identifies top-performing regions and dealers
* Highlights sales trends and seasonal patterns
* Provides comparison across different car categories
* Helps in data-driven decision-making for sales strategy
